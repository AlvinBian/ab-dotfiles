# Slack mrkdwn 格式規範

## 禁止事項

- 禁止使用 `**文字**` 粗體（改用 `*文字*`）
- 禁止使用 `[文字](url)` 連結（改用 `<url|文字>`）
- 禁止使用 `---` 分隔線（改用空行）
- 禁止使用 `## 標題`（改用 `*粗體*` 單獨一行）
- 禁止使用 `~~刪除線~~`（改用 `~文字~`）
- 禁止在連結前加標籤前綴（`*📖 文件：* <url|文字>` → 直接 `<url|文件名稱>`）
- 格式符號前後不可有空白：`* 文字 *` 無效，要寫 `*文字*`

## 語法對照

| 效果 | 正確 | 錯誤 |
|------|------|------|
| 粗體 | `*文字*` | `**文字**` |
| 斜體 | `_文字_` | — |
| 刪除線 | `~文字~` | `~~文字~~` |
| 連結 | `<url\|顯示文字>` | `[文字](url)` |
| 提及用戶 | `<@USERID>` | `@名字` |
| 頻道 | `<#CHANNELID\|name>` | `#name` |

## 結構規範

- 段落之間留空行
- 清單用 `•` 或 `-` 開頭，不支援巢狀縮排
- 引言：`> 文字`（只支援單層）
- 程式碼：`` `inline` `` 或 ` ``` 多行 ``` `（區塊內 mrkdwn 失效）
- 訊息超過 400 字建議改用 Canvas 或分段

## 提及規範

- 特定用戶：`<@USERID>`（必須是真實 ID，`@名字` 不觸發通知）
- 頻道在線成員：`<!here>`
- 頻道所有成員：`<!channel>`（謹慎使用）
