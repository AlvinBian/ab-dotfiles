# Draft Slack Message

## Step 1 — 收集資訊

詢問（若未提供）：
1. 訊息主題 / 內容
2. 類型：`公告` / `進度更新` / `技術分享` / `緊急告警` / `自由格式`
3. 目標頻道或對象（選填）

## Step 2 — 生成草稿

### 格式規則

| 項目 | 正確 | 錯誤 |
|------|------|------|
| 粗體 | `*文字*` | `**文字**` |
| 連結 | `<url\|顯示文字>` | `[文字](url)` |
| 分隔線 | 空行 | `---` |
| 標題 | `*粗體*` 單獨一行 | `## 標題` |
| 提及 | `<@USERID>` | `@名字` |

### 模板

**公告**
```
:loudspeaker: *【{標題}】*

{內容}

*時間：* {日期}
*負責人：* <@USERID>
```

**進度更新**
```
*📦 {主題} 更新*

• ✅ 已完成：{項目}
• 🔄 進行中：{項目}
• ⏳ 待處理：{項目}
```

**緊急告警**
```
:rotating_light: *[{等級}] {問題}*

*影響：* {說明}
*狀態：* {調查中/已修復}

處理人：<@USERID>
```

**技術分享**
```
:books: *【{標題}】*

_{說明}_

*重點*
• {要點}

<url|文件>
```

## Step 3 — 後續選項

1. 直接複製使用
2. 修改後再發
3. 透過 Slack MCP 發送（`slack_send_message`）

> ⚠️ 發送時用 `channel_id`（不是 `channel`）。若失敗，改用 `slack_schedule_message` 設定 5 分鐘後。
